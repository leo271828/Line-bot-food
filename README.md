# 肚子餓餓
你是否每天都會有「今天要吃甚麼?」這個問題呢?身為成大生，周圍的餐廳選擇必定是非常多，因此常常造成選擇困難\
但現在我們的「肚子餓餓」能夠為你解決這個問題！

## 使用方法
* 基本輸入
	* `fsm`
		* 秀出當下 FSM
	* `back`
		* 回到初始狀態
	* `menu`
		* 開始動作！
- Menu
    - 美食推薦
        
        輸入下面三個資訊：
        
        1. 種類
        2. 價位
        3. 地點
    - 美食清單
        - 新增功能
        - 查看功能
            - ALL
            - 5 家
